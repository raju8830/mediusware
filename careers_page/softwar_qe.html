Software QA Engineer

Dhaka, Bangladesh

Job Context

We are looking for a Quality Assurance (QA) engineer to develop and execute exploratory and automated tests to ensure our softwareproduct quality.QA engineer responsibilities include designing and implementing tests, debugging, and defining corrective actions. You will also review system requirements and track quality assurance metrics (e.g. defect densities and open defect counts.) Our ideal candidate will be responsible for conducting tests before product launches to ensure software runs smoothly and meets client needs while being cost-effective. Therefore, you should monitor all stages of software development to identify and resolve system malfunctions to meet quality standards

Job Requirements:
  • Experience with automation tools like Selenium, Robot Framework, TestNG, AutoIT, etc
  • 2+ years of experience in the relevant area
  • PHP experience is bonus.
  • Possess an investigative attitude while testing the product.
  • Ability to visualize real-time business situations and scenarios.
  • Able to capture and transform business requirements into detailed test scenarios.
  • Keen to learn browser internals and data communication implementations.
  • A strong desire to learn and develop technical skills.
  • Ability to plan tasks and manage time effectively.
  • Excellent written and verbal communication skills.
  • Ability to understand the strengths and weaknesses of teammates in order to work with them constructively.
  • Results-oriented R&D skills and proactive attitude
Job Responsibilities:
  • Solid knowledge of SQL Scripting
  • Hand on experience in project testing using Jira, Blackbox, and Whitebox.
  • Understand the client's requirements and determine the testing scope.
  • Breakdown tasks into major test areas and estimate the time.
  • Determine and define test strategies for all sorts of applications.
  • Able to prepare test cases, checklists, traceability matrix as per requirements.
  • Execute tests or write test code to verify enhancements and bug fixes
  • Report bugs/defects with appropriate details.
  • Prepare all sorts of test documents (test plan, test report, release notes, etc.
  • Retrieve the latest code from the appropriate branch.
  • Submit changes to the appropriate branch with comments.
  • View the repository log and understand where any specific modification has been made.
  • Merge source code changes made by other team members.
  • Perform R&D to identify the scope of test ideas that may scale up the output level
  • Review all sorts of tasks of team members.
  • Communicate with Dev lead/ manager to improve development processes and resolve problems.
Educational Requirements:

Minimum Graduation in CSE. Any professional certificate in JIRA,
Whitebox, and Blackbox will be an added advantage.

Experience Requirments
  • At least 2 year(s)
  • The applicants should have experience in the following area(s): JIRA, Blackbox, Whitebox
