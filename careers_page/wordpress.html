WordPress Developer

Dhaka, Bangladesh

Job Context

We are looking for motivated and innovative WordPress Developer to join our team who will be responsible for developing WordPress themes while replying to support tickets and troubleshooting theme issues.

Job Requirements:
  • Proven work experience as a WordPress Developer.
  • WooCommerce skill is Mandatory.
  • Knowledge of front-end technologies including CSS3, JavaScript, HTML5, and jQuery.
  • Knowledge of code versioning tools including Git, Mercurial, and SVN.
  • Positive and upbeat attitude with the ability to learn quickly and problem solving.
  • Be open to receiving feedback and constructive criticism.
  • Handling multiple projects at the same time.At least 2 year(s)
Job Responsibilities:
  • Developing WordPress themes for marketplaces like ThemeForest, and WordPress.org
  • Maintain existing plugins and fix bugs and add new features.
  • Integrating and setting up plugins like WPML, WooCommerce, Membership, etc.
  • Optimizing page speed of WordPress sites
  • Common WordPress Troubleshooting and Debugging
  • Solid understanding of WordPress Action/Hooks and its API's
  • Solid understanding of WordPress Theme unit test
  • Understanding of WP architecture and knowledge on performing database and code optimization for WordPress
  • A thorough understanding of cross-browser compatibility issues
  • Strong understanding of responsive web design techniques
  • Working with version control systems such as GIT
Educational Requirements:

Bachelor’s degree in any discipline.
